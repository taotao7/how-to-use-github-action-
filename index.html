⛷ Git Action

CI/CD

持续集成CI(Continuous Integration)

持续交付CD(Continuous Delivery)

持续部署CD(Continuous Deployment)

CI 持续集成

合并，测试，代码健康检查...

确保统一环境，完整性测试，防止破坏性更改,如果发现有问题可以用ci进行修复或者提醒维护者,aone的例子，代码健康等等

CD 持续交付

在CI通过测试，校验等后自动合并代码或者构建编译等操作

更加高效的自动化工作流程，减少重复工作

CD 持续部署

作为交付的延续工作，可以自动把通过校验的代码部署到服务器

更加高效的自动化工作流程，减少重复工作
